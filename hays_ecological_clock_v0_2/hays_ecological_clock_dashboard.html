v0.2 Live-Pull Setup

This package now includes Hays County boundary GeoJSON, clipped Level IV ecoregions, PRISM/USA-NPN/iNaturalist/GBIF/eBird script templates, and a credential-safe configuration file.

  • PRISM: 4 km daily climate, 2020-present, 1991-2020 normals.
  • USA-NPN: 2015-present phenology setup for leaves, flowers, fruits, ripe fruits, and seed drop.
  • iNaturalist/GBIF/eBird: public/live observation scaffolding for Hays County.
